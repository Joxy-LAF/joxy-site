Joxy

A look-and-feel for Java Swing applications to let them look like KDE applications with the Oxygen style.

What is Joxy?

The KDE desktop environment uses the Oxygen theme by default. By default, Java applications don't follow this theme at all. The Joxy Look and Feel is a "skin" to let Java Swing applications look like the Oxygen theme. Joxy is not an application itself, but a skin for other programs.

Theming

Joxy tries to detect which KDE theme you use, and uses that to style Java programs accordingly.

Is it stable?

Joxy is still in alpha status, so bugs are to be expected. However, many programs should work without problems.
